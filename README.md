<h1 align="center">interactive-cv</h1>

**Open-source-шаблон персонального интерактивного сайта-резюме с двумя AI-функциями:**
1. **AI-ассистент для рекрутёра** — отвечает на вопросы по фактам из CV в реальном времени (Server-Sent Events) и с защитой от галлюцинаций.
2. **AI-генератор откликов** — создаёт адаптированный CV и cover letter прямо из текста вакансии.

Любой разработчик может развернуть такой сайт под своё CV: меняется только контент (markdown) и домен, код остаётся прежним.

Живой пример сервиса, собранного из этого шаблона: **[cv.libera.pro](https://cv.libera.pro)**.

## Что это

Обычное резюме — статичный PDF. Этот проект превращает CV в интерактивный сайт с двумя AI-механиками, которые работают в связке:

- 🤖 **AI-ассистент (для рекрутёра)** — рекрутёр задаёт вопросы в чат («какие проекты на Flutter?», «делал ли DevOps?»), а модель (glm-5.2 от Z.ai) отвечает **по фактам из CV** в реальном времени (Server-Sent Events) и с защитой от галлюцинаций. Диалог сохраняется — при перезагрузке контекст не теряется
- ⚡ **AI-генератор откликов (для владельца)** — вставляете текст вакансии или фриланс-заказа → LLM генерирует адаптированный CV и cover letter под конкретную позицию → редактируете в split-редакторе с live-preview → публикуете с короткой ссылкой
- 🚀 **Фриланс-режим** — отдельный тип откликов (vacancy/freelance), бюджет, конкурс, дедлайн, рейтинг звёздами, ТЗ из PDF (загрузка → парсинг → в промпт), отдельный freelance-промпт
- 📄 **PDF-экспорт** — скачивание CV в PDF (Unicode, кириллица) для отклика на FL.ru и другие площадки
- 🕸 **Граф знаний** — визуализация связей проектов и технологий
- 🔗 **Короткие ссылки с TTL** — персональная ссылка под каждую вакансию с аналитикой открытий (клики, уникальные посетители с именами)
- 📊 **Админ-панель** — управление откликами (вакансии + фриланс), собеседованиями, чатами HR и аналитикой из браузера
- ⚙ **Настройки в БД** — мастер-CV, README и промпты редактируются через админку (AI-правка CV по инструкции), без деплоев
- 💬 **Просмотр HR-чатов** — все диалоги рекрутёров с именами посетителей («Крепкий Кабан», «Деловой Барсук»), привязка к откликам

## Архитектура (два репозитория)

| Репозиторий | Стек | Назначение |
|-------------|------|-----------|
| [cv-backend](https://github.com/interactive-cv/cv-backend) | Python, FastAPI, PostgreSQL | REST API, AI-промпты (чат + генерация CV + freelance + правка), PDF-экспорт, трекинг посетителей, сессии чата, аналитика |

## Технологические решения, которые стоит увидеть

### AI-интеграция (ключевая фишка проекта)

- **RAG без векторной БД** — для объёма одного CV весь текст подаётся как контекст в системном промпте (дешевле и проще pgvector). Используется и в чате, и в генераторе откликов.
- **Защита от галлюцинаций** — жёсткий промпт прямо запрещает модели выдумывать опыт/метрики/работодателей + e2e-тесты против реальной glm-5.2 на провокационных запросах (модель не подтверждает выдуманных работодателей)
- **Стриминг ответа** — токены идут в браузер по мере генерации (SSE/ReadableStream), как в ChatGPT, с typing-индикатором
- **AI-генерация CV из вакансии** — вставка текста вакансии + выбор проектов → LLM создаёт адаптированный CV и cover letter, переупорядочивая и подсвечивая релевантное, но не искажая факты
- **Graceful degradation** — при сбое LLM чат не падает (500), а отдаёт контакты; остальной сайт работает

### Инженерные решения

- **Кросс-БД типы** — тесты на in-memory SQLite, прод на PostgreSQL с нативными `jsonb`/`uuid` без изменения кода
- **Атомарный инкремент** счётчика кликов без race condition через SQL `UPDATE ... WHERE`
- **Авто-TLS** — собственный nginx-образ с Let's Encrypt, сертификаты получаются и продлеваются автоматически

## Развёртывание

Прод-окружение — собственный VPS с Docker Compose. Весь стек (nginx + FastAPI + Next.js + PostgreSQL) поднимается одной командой. Подробности — в README обоих репозиториев.
